# MDWX - 微信公众号专业排版工具

专为高度审美要求的创作者打造的 Markdown 编辑器，支持一键渲染、自定义美学主题、代码高亮和外链转注脚。让写作回归内容本身。

## 核心特性

- **完美适配**: 专为微信公众号优化的渲染引擎。
- **美学主题**: 内置 Tickmao、Optima 等多款高设计感主题。
- **代码高亮**: 基于 Highlight.js 的完美代码块渲染。
- **外链转注**: 自动将文章中的链接转换为文末参考文献索引。
